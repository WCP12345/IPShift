## Quickly switch between saved IP configurations on Windows.

IPShift is a lightweight Windows utility designed to quickly switch between saved network IP configurations.

Instead of manually navigating through Windows network adapter settings every time you move between networks, IPShift allows you to store multiple IP profiles and apply them instantly.

Designed for engineers who frequently switch between machine networks, office networks, and device subnets.

This is especially useful for engineers, IT staff, and anyone regularly connecting to devices on different subnets.

## Features

• Save and manage multiple IP configuration profiles  
• Quickly switch between **static IP settings and DHCP**  
• Apply profiles to specific network adapters  
• **Device ping / status monitoring** – add commonly used devices to a profile and quickly check their online status  
• **Shared profiles** – link to a shared folder so teams can access and use the same profiles  
• **Light and dark theme support**  
• Simple and lightweight interface designed for fast use

## Typical Use Cases

• Industrial automation / PLC networks  
• Switching between office and machine networks  
• Device testing on different subnets  
• Field engineers connecting to multiple systems
